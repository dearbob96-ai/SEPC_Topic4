# 🌍 淡江大學與淡水區國際友善空間體檢報告展示平台

## 📖 專案簡介

本專案為《社會議題探索暨實踐課程》專題四「文化馬賽克與文化融爐」的專屬網頁化報告展示平台。

專案旨在將傳統的學術觀察報告轉換為具備現代感、易讀性高的互動式網頁。內容涵蓋了**任務一（淡江大學國際友善校園體檢）**與**任務二（新北市淡水區國際友善城市體檢）**。透過此平台，讀者可以輕鬆瀏覽評估框架、校園與城市各項指標的系統性分析，並透過「圖片藝廊」直觀地檢視實地田野調查的影像紀錄。

## ✨ 核心特色

*   **單頁式應用架構 (SPA)**：無須重新載入頁面即可流暢瀏覽各章節內容。
*   **側邊欄動態追蹤 (Scrollspy)**：閱讀長篇報告時，側邊導覽列會自動高亮當前閱讀章節，提供極佳的導航體驗。
*   **響應式設計 (RWD)**：完美支援桌上型電腦、平板與手機瀏覽。
*   **田野影像藝廊**：專設第四章節展示實地勘查照片，將文字論述轉化為具體的視覺證據。

## 🛠️ 技術堆疊

*   **HTML5 / 語意化標籤**：建構清晰的文本結構。
*   **Tailwind CSS (CDN)**：快速構建現代化、一致性的 UI 介面，免去繁瑣的 CSS 撰寫。
*   **Vanilla JavaScript**：處理側邊欄滾動監聽與介面互動邏輯。

## 🚀 如何使用與部署

本專案為純前端靜態網頁，無需設定任何後端伺服器或資料庫。

1.  **下載專案**：將專案原始碼（HTML 檔案）下載至您的電腦中。
2.  **開啟網頁**：直接對 HTML 檔案點擊右鍵，選擇使用 Chrome、Edge 或 Safari 等現代瀏覽器開啟即可預覽。
